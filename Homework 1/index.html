Introduction to HTML

HTML (HyperTextMarkupLanguage) is the code that is used to structure a webpage and its content.For example content could be structured within paragraphs,a list of bulleted points or using images and data tables.

PS: It was invented by physicist Tim Berners-Lee,The first version was published in 1993

So what is HTML?

HTML is a markup language that defines the structure of your content.It consists of elements which you use to enclose,or wrap different parts of your content to make it appear or act a certain way.For example if we wanted a line to stand by itself we could specify that it is a paragraph by enclosing it in paragraph tags.

Anatomy of an HTML element

The main parts of an HTML element are as follows;

  1. The opening tag:This consists of the name of the element(e.g p), wrapped in opening and closing angle brackets. This states where the element begins or starts to take effect.
  2. The closing tag:This is the same as the opening tag,except that it includes a forward slash before the element name. This states where the element ends-In this case where the paragraph ends.

Summary

HTML is a markup language used to describe the structure and content of a webpage,it says what content should be on a webpage.An HTML document is made up of a series of elements.Some elements we covered in this lesson are...
